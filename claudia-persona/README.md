## Memory System Architecture

Claudia uses a 4-layer memory system:

### Layer 1: Hot Memory (`memory/executive/hot/`)
Current priorities, dashboard, alerts

### Layer 2: PARA (`memory/executive/para/`)
- **Projects**: Active work with briefs
- **Areas**: Ongoing responsibilities
- **Resources**: Reference materials
- **Archive**: Completed work

### Layer 3: Timeline (`memory/executive/timeline/`)
Daily logs with automatic date stamping

### Layer 4: Intelligence (`memory/executive/intelligence/`)
- **Patterns**: Recurring solutions
- **Relationships**: Contact context
- **Market**: Industry insights
