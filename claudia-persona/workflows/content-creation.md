# Content Creation Workflow

## Phase 1: Strategy & Planning

**Claudia's Actions:**
1. Define content goals (awareness, engagement, conversion)
2. Identify target audience and personas
3. Research topics and keywords
4. Create content calendar

**Output:** Content strategy document + editorial calendar

## Phase 2: Research

**Claudia's Actions:**
1. Gather source materials
2. Interview subject matter experts
3. Review competitor content
4. Identify unique angles

**Output:** Research brief with sources and angle

## Phase 3: Creation

**Claudia's Actions:**
1. Create detailed outline
2. Write first draft
3. Review and self-edit
4. Incorporate feedback

**Output:** Polished content piece

## Phase 4: Production

**Claudia's Actions:**
1. Format for target platform
2. Create supporting visuals
3. Write meta descriptions
4. Prepare social snippets

**Output:** Publication-ready assets

## Phase 5: Distribution

**Claudia's Actions:**
1. Schedule across channels
2. Set up email campaign
3. Coordinate with influencers
4. Monitor initial performance

**Output:** Distributed content

## Phase 6: Analysis

**Claudia's Actions:**
1. Track performance metrics
2. Gather qualitative feedback
3. Identify improvement areas
4. Update content strategy

**Output:** Performance report + learnings
